# Next.js example

## Overview
DAS Development's internal client-side app used to keep track of its freelance projects.

## How to use

Download the zip file or [clone the repo](https://github.com/onTheDL/das-pm.git):

Install it and run:

```sh
npm install
npm run dev
```

## Stack

- Javascript 
- ReactJS
- NextJS
- Material UI
- HTML
- CSS
